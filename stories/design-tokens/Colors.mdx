# Color Tokens

All color tokens used in the design system. Colors are defined using OKLCH color space for better perceptual uniformity and are automatically adapted for light and dark modes.

## Usage Examples

### Tailwind CSS Classes

```tsx
// Background colors
<div className="bg-primary">Primary background</div>
<div className="bg-secondary">Secondary background</div>
<div className="bg-accent">Accent background</div>

// Text colors
<p className="text-primary">Primary text</p>
<p className="text-muted-foreground">Muted text</p>

// Border colors
<div className="border border-border">Bordered element</div>
```

### CSS Variables

```css
.custom-element {
  background-color: var(--color-primary);
  color: var(--color-primary-foreground);
  border: 1px solid var(--color-border);
}

.custom-element:focus {
  outline: 2px solid var(--color-ring);
}
```

## Theme Switching

All colors automatically adapt when the `.dark` class is present on the root element. The design system handles this automatically based on user preference.
